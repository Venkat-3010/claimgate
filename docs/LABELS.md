# GitHub labels

Suggested labels for this repository:

| Label | Use |
| --- | --- |
| `bug` | Incorrect behavior |
| `enhancement` | New capability |
| `good first issue` | Onboarding-sized tasks (see CONTRIBUTING) |
| `adapter` | Gate adapter work |
| `cli` | CLI UX |
| `mcp` | MCP server (keep ≤3 tools) |
| `docs` | Documentation |
| `ci` | GitHub Actions / pipelines |
| `false-green` | Detection of misleading pass signals |
| `security` | Security-sensitive changes |
